# Neex Creative — Website Portfolio

This repository contains the source code for the **Neex Creative** portfolio website (static site).

## Live Website
- GitHub Pages: https://neexcreativeio.vercel.app/

## Project Structure
- `index.html` — Main page
- `about.html` — About page
- `book.html` — Book a Call page (Calendly embed / link)
- `navbar.html` — Navbar markup (loaded dynamically)
- `navbar-loader.js` — Injects `navbar.html` into pages
- `style.css` — Global styles
- `script.js` — Front-end interactions (if applicable)
- `assets/img/` — Images and assets (projects, backgrounds, etc.)

## How to Run Locally
You can open the site with any local server. Example (VS Code):
1. Install **Live Server** extension
2. Right-click `index.html` → **Open with Live Server**

## How to Update the Website
Recommended workflow:
1. Edit the website locally (VS Code)
2. Commit changes
3. Push to GitHub (`main` branch)
